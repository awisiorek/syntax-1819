## Themen:

- **Syntax: Linguistische und formale Grundlagen**
- **Parsing: CFG-Parsingalgorithmen und Unifikationsparsing**
- **Statistisches Parsing: PCFGs und Dependency Parsing**
- **Partielles Parsing: Chunking und reguläre Grammatiken**
